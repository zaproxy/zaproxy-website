# OWASP PTK Active Scan Rule

This add-on provides a single active scan rule, **PTK Scan Rules** , that drives browsers against the target host so that PTK's in-browser engines (SAST, IAST, DAST) run as the browsers traverses the application.


It is at an early stage and more changes are planned, and for that reason is is disabled by default.
It can be enabled via the [OWASP PTK Options](/docs/desktop/addons/owasp-ptk/ptk-options/)
or via the command line: `-config ptk.activescan.rule.enabled=true`)


See also: [OWASP PTK](/docs/desktop/addons/owasp-ptk/), [OWASP PTK Options](/docs/desktop/addons/owasp-ptk/ptk-options/)
