# Fuzz AI Files

Provides a set of files for fuzzing AIs (for example via an API), based on a variety of models such as [Artificial Intelligence Resilience Maturity Model (AI-RMM)](https://osf.io/kb8a5/).


The following files are included, and will appear as ZAP "Fuzzing Files" payloads.

* [Extract Training Data](/docs/desktop/addons/fuzzai-files/extract-training-data/)
* [Extract Model Information](/docs/desktop/addons/fuzzai-files/extract-model-info/)
* [Exploit-Model-Memory](/docs/desktop/addons/fuzzai-files/exploit-model-memory/)
* [Test Edge Cases](/docs/desktop/addons/fuzzai-files/test-edge-cases/)
