# Common Library

A library for add-ons.


The following context menu items are provided for the Sites tree:

* **Get Info** - displays a summary of that node's subtree, including:
    * The total number of nodes in the subtree.
    * The timestamp of the most recently added node.
    * A breakdown of the request sources (e.g. proxy, authentication) found in the subtree.
* **Expand/Collapse Node and Children** - Recursively expands or collapses the selected node and all of its children.
