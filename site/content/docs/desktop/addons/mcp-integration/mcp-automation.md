# MCP Automation Framework Support

This add-on supports the [Automation Framework](/docs/desktop/addons/automation-framework/).


The following jobs are available:

* [mcp-config](/docs/desktop/addons/mcp-integration/mcp-automation-config/) - Configure the MCP server (port, security key, enabled state, etc.)
* [mcp-import](/docs/desktop/addons/mcp-integration/mcp-automation-import/) - Import an external MCP server into ZAP's history and sites tree
