ZAP  supports the following scripting languages:

| Language / Link                                         | Included   | Notes |
| ------------------------------------------------------- | ---------  | ----- |
| [JavaScript](/docs/desktop/addons/graalvm-javascript/)  | By Default | Based on the GraalVM JavaScript engine. |
| [Zest](/docs/desktop/addons/zest/)                      | By Default | A graphical security scripting language, ZAPs macro language on steroids. |
| [BeanShell](/docs/desktop/addons/bean-shell/)           | Optional   | A BeanShell Console with limited capabilities. |
| [Groovy](/docs/desktop/addons/groovy-support/)          | Optional   | Based on Groovy 5.0.3. |
| [Kotlin](/docs/desktop/addons/kotlin-support/)          | Optional   | Based on Kotlin 1.3.72. |
| [Python](/docs/desktop/addons/python-scripting/)        | Optional   | Based on Jython 2.7.2. |
| [Ruby](/docs/desktop/addons/ruby-scripting/)            | Optional   | Based on JRuby 1.7.4. |
